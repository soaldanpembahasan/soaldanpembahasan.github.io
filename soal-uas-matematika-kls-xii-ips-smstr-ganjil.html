2. tlong di jawab matematika kls 8 smstr 1 no. 8


Un = a + b (n-1)
U4 = a + 3b = 65 ...(1)
U8 = a + 7b = 93 ...(2)

Dit: S8 = ...

Penyelesaian:
Sn = (n/2) (a + Un)
S8 = (8/2) (a + U8)

Untuk mencari nilai a, eliminasi pers (1) dan (2)
a + 3b = 65
a + 7b = 93
----------------- -
-4b = -28
b = 28/4 = 7

Dari per (1)
a + 3b = a + 3(7) = a + 21 = 65
a = 65 - 21 = 44

S8 = (8/2) (a + U8)
= 4 ( 44 + 93)
= 4 x 137
= 548
