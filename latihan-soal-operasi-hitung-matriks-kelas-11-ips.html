20. hitung a = {3,2,3,2,5,4,3,0,1 } dan b = {1,2,1,2,3,1,4,5,1} dan hitunglah matriks c dengan operasi menjumlahkan matriks a dan b ,juga matriks d dengan operasi pemgurangan ?​


Jawaban:

Matriks a = {3,2,3,2,5,4,3,0,1}

Matriks b = {1,2,1,2,3,1,4,5,1}

Matriks c (penjumlahan) = a + b

= {3+1, 2+2, 3+1, 2+2, 5+3, 4+1, 3+4, 0+5, 1+1}

= {4, 4, 4, 4, 8, 5, 7, 5, 2}

Matriks d (pengurangan) = a - b

= {3-1, 2-2, 3-1, 2-2, 5-3, 4-1, 3-4, 0-5, 1-1}

= {2, 0, 2, 0, 2, 3, -1, -5, 0}

Sebagai catatan, operasi penjumlahan dan pengurangan pada matriks dilakukan pada elemen yang bersesuaian. Sehingga untuk matriks c dan d, elemen pertama dari a dijumlahkan/dikurangkan dengan elemen pertama dari b dan seterusnya.
