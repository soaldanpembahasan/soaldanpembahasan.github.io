9. diketahui kubus KLMN. PQRS dengan panjang rusuk 13 cm. tentukan panjang KM, KS, dan KR.​bantuin kak​


Jawaban:

Kubus KLMN.PQRS dalam soal merupakan salah satu soal penerapan teorema pythagoras.

Pythagoras menyatakan bahwa : “Untuk setiap segitiga siku-siku berlaku kuadrat panjang sisi miring (Hipotenusa) sama dengan jumlah kuadrat panjang sisi siku-sikunya.”

Jika sisi-sisi pada segitiga siku-siku kita beri nama a, b, dan c. Dimana a dan b merupakan sisi sisi yang mengapit sudut siku-siku dan c merupakan sisi miring, atau sisi terpanjang, maka berlaku ↓

a² + b² = c²

Pembahasan

Diketahui kubus KLMN.PQRS.

Panjang rusuk KL = LM = MN = NK = 13 cm.

Untuk menentukan panjang diagonal bidang KM kita menggunakan teorema Pythagoras, sehingga

KM² = KL² + LM²

KM² = 13² + 13²

KM² = 169 + 169

KM² = 338

KM = √338

KM = √(169 x 2)

KM = √169 x √2

KM = 13√2

Jadi, panjang diagonal bidang KM adalah 13√2 cm.

Untuk menentukan panjang diagonal ruang PM, kita menggunakan teorema Pythagoras, sehingga

PM² = PK² + KM²

PM² = 13² + (13√2)²

PM² = 169 + (169 x 2)

PM² = 169 + 338

PM² = 507

PM = √507

PM = √(169 x 3)

PM = √169 x √3

PM = 13√3

Jadi, panjang diagonal ruang PM adalah 13√3 cm.

14. Massa jenis rata-rata = p = KETIDAKPASTIAN RELATIF PENGUKURAN BERULANG : Δρ = % KR = % KR = NX-Y N-1 PERSENTASE KETIDAKPASTIAN RELATIF PENGUKURAN BERULANG % KR 4x100% Р N —×100% % Y = (Σp)²=​


Jawaban:

Tampaknya Anda memberikan rumus-rumus terkait ketidakpastian relatif dalam pengukuran berulang. Mari kita urai lebih lanjut:

- Massa jenis rata-rata (p̄) adalah hasil pengukuran berulang yang dinyatakan dalam persentase ketidakpastian relatif pengukuran berulang (% KR).

- Ketidakpastian relatif pengukuran berulang (% KR) dihitung sebagai:

% KR = Δρ / p̄ x 100%

- Δρ adalah ketidakpastian (uncertainty) dalam pengukuran tersebut.

- KETIDAKPASTIAN RELATIF PENGUKURAN BERULANG (% KR) dapat dinyatakan sebagai:

% KR = (Σp) / (p̄ * (N-1)) x 100%

Di sini, Σp adalah jumlah ketidakpastian dalam pengukuran berulang.

- N adalah jumlah pengukuran yang diambil.

- % Y adalah persentase ketidakpastian relatif pengukuran berulang yang dihitung sebagai:

% Y = (Σp)² / (p̄ * (N-1)) x 100%

Dalam konteks ini, rumus-rumus tersebut digunakan untuk mengukur ketidakpastian dalam pengukuran berulang dan persentase ketidakpastian relatifnya. Ini adalah bagian dari analisis ketidakpastian dalam pengukuran dan eksperimen untuk mengevaluasi sejauh mana hasil pengukuran dapat diandalkan.
