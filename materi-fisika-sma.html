No. 20
---------
Diket : 
Transformator (TRAFO)
jumlah lilitan primer, Np = 150 lilitan
Jumlah lilitan sekunder, Ns = 30 Lilitan
Spesifikasi Lampu,
Tegangan tertulis dilampu, V = 220 V
Daya tertulis dilampu, P = 30 Watt
Efisiensi trafo, η = 90%

Tanya : 
Arus pada kumparan Primer, Ip = __?

Jawab :

Step - 1
----------
Cari dulu daya sekunder, Ps = __? dan Tegangan Sekunder, Vs = __?
Asumsi Lampu Nyala NORMAL, maka :
A] daya sekunder trafo sama dengan daya yang tertulis pada lampu sebesar, Ps = P = 30 watt
B] tegangan sekunder trefo sama dengan tegangan yang tertulis pada lampu, Vs = V = 220 V

Step - 2
----------
Cari tengan pada kumparan primer, Vp = ___?
Gunakan rumus :
Vp / Vs = Np / Ns
Vp / 220 = 150 / 30
Vp / 220 = 5
Vp = 5 . 220
Vp = 1100 Volt

Step - 3
-----------
Cari Arus pada kumparan Primer, Ip = __?
Gunakan rumus :
Ps = η.Pp
Ps = η.Vp. Ip
30 = 90%. 1100. Ip
30 = 0,9. 1100. Ip
30 = 990. Ip
Ip = 30 / 990
Ip = 1/33 Ampere
Ip = 0,0303 A

Diket : 
Rangkaian R dan C seri 
Hambatan Resistor, R = 3000 ohm
frekuensi, f = 100/6π Hz
Tegangan pada resistor, VR = 30 Volt

Kapasitas kapasitor, C = __?

Jawab :

--------
Cari hambatan kapasitor atau reaktansi kapasitif, XC = __?

VR / R = VC / XC
 30 / 3000 = 60 / XC

C = 1/(200000)
C = 1/(2. 10⁵)

7. >>Fisika kelas 10 SMA<<_Ringkasan_ materi tentang gerak jatuh bebas, gerak melingkar, gerak parabola, dan gerak lurus Tolong bantuu :'Jangan ngasal oke :') ​


Penjelasan:

1. Gerak Jatuh Bebas adalah gerak yang lintasannya lurus dengan arah ke bawah tanpa kecepatan awal (Vo = 0). Benda yang mengalami gerak jatuh bebas juga mendapatkan percepatan yang besarnya sama dengan percepatan gravitasi bumi (g).

Gerak jatuh bebas juga termasuk gerak dipercepat beraturan sehingga pada gerak jatuh bebas berlaku rumus :

Vt = Vo + gt

h = Vo.t + 1/2 gt²

Oleh karena gerak jatuh bebas tanpa kecepatan awal (Vo = 0), maka rumus di atas dapat berubah menjadi sebagai berikut :

Vt = g.t

Vt² = 2gh

h = 1/2 gt²

2. Gerak Parabola merupakan gabungan antara gerak lurus yang arahnya mendatar (sumbu-x) dan gerak lurus yang arahnya vertikal (sumbu-y). Cth gerak parabola antara lain, gerak peluru yang lepas dari senapan, gerak bola golf, bola bisbol yang dilempar, bola sepak yang ditendang, dan bola basket yang dilempar.

3. Gerak Melingkar yaitu gerak dengan lintasan berupa lingkaran. Lintasan melingkar dapat berupa satu lingkaran penuh atau lebih dan dapat pula berupa sebagian busur lingkaran.

4. Gerak Lurus.. Dalam konsep gerak lurus dikenal beberapa istilah, antara lain jarak, perpindahan, kelajuan, kecepatan, dan percepatan.

Jarak adalah panjang lintasan yang ditempuh suatu benda selama bergerak dalam selang waktu tertentu.Perpindahan adalah perubahan posisi awal terhadap posisi akhir dalam selang waktu tertentu.Kelajuan adalah jarak yang ditempuh benda tiap satuan waktu. Kelajuan juga termasuk ke dalam besaran skalar.Kecepatan adalah perpindahan yang dialami benda tiap satuan waktu. Kecepatan termasuk ke dalam besaran vektor.Percepatan adalah perubahan kecepatan benda tiap satuan waktu

9. Fisika Kelas XI SMA Materi : Keseimbangan dan Gerak Rotasi Rules : ★ Dilarang SPAM ★ Gunakan Cara Lengkap


~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Fisika Kelas XI SMA
Materi : Keseimbangan dan Gerak Rotasi

Nomor 3
-------------
Diket :
tiga partikel dengan besar massa nya adalah :
m1 = 1 kg
m2 = 2 kg
m3 = 3 kg

Terletak pada titik koordinat :
partikel m1 (3,6) artinya : x1 = 3 dan y1 = 6
partikel m2 (0,6) artinya : x2 = 0 dan y2 = 6
partikel m3 (3,0) artinya : x3 = 3 dan y3 = 0

Tanya :
Letak koordinat pusat massa, 
Xpm = __ ? dan Ypm = __?

Jawab :

Xpm = (x1.m1 + x2.m2 + x3.m3) / (m1 + m2 + m3)
Xpm = (3.1 + 0.2 + 3.3) / (1 + 2 + 3)
Xpm = (3 + 0 + 9) / (6)
Xpm = (12) / (6)
Xpm = 2 

dan  

Ypm = (y1.m1 + y2.m2 + y3.m3) / (m1 + m2 + m3)
Ypm = (6.1 + 6.2 + 0.3) / (1 + 2 + 3)
Ypm = (6 + 12 + 0) / (6)
Ypm = (18) / (6)
Ypm = 3

Jadi koordinat pusat massa dari ketiga partikel adalah (2,3)



Nomor 4
-----------
Diket :
Dari gambar grafik bisa kita peroleh :
empat partikel dengan massa, 
m1 = 1 x 10⁻³ kg = 1 gram
m2 = 2 x 10⁻³ kg = 2 gram
m3 = 3 x 10⁻³ kg = 3 gram
m4 = 3 x 10⁻³ kg = 3 gram

dan Terletak pada titik koordinat :
partikel m1 (6,2) artinya : x1 = 6 dan y1 = 2
partikel m2 (6,4) artinya : x2 = 6 dan y2 = 4
partikel m3 (2,4) artinya : x3 = 2 dan y3 = 4
partikel m4 (2,2) artinya : x4 = 2 dan y4 = 2

Tanya :
Letak koordinat pusat massa, 
Xpm = __ ? dan Ypm = __?

Jawab :

Xpm = (x1.m1 + x2.m2 + x3.m3 + x4.m4) / (m1 + m2 + m3 + m4)
Xpm = (6.1 + 6.2 + 2.3 + 2.3) / (1 + 2 + 3 + 3)
Xpm = (6 + 12 + 6 + 6) / (9)
Xpm = (30) / (9)
Xpm = (10) / (3)
Xpm = 10/3

dan 

Ypm = (y1.m1 + y2.m2 + y3.m3 + y4.m4) / (m1 + m2 + m3 + m4)
Ypm = (2.1 + 4.2 + 4.3 + 2.3) / (1 + 2 + 3 + 3)
Ypm = (2 + 8 + 12 + 6) / (9)
Ypm = (28) / (9)
Ypm = 28/9

Jadi letak pusat massa dari sistem ke empat partikel itu adalah 
(10/3 , 28/9)
