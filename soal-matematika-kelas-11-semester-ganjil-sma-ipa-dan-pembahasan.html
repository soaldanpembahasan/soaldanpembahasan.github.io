17. tolong bantu kakak-kakak, ini soal kelas 10 semester ganjil matematika​


penyelesaian SPLDV dari x + 3y = 5 dan x + y = 3 adalah {2, 1}

penyelesaian SPLDV dari 4x - 3y = 7 dan x + 2y = -1 adalah {1, - 1}

Pembahasan

a. x+y = 3 diubah menjadi x = 3-y

sehingga

(3-y)+3y = 5

-y+3y = 5-3

2y = 2

[tex] \sf y= \dfrac {2}{2} [/tex]

y = 1

sehingga x

x+y = 3

x+1 = 3

x = 3-1

x = 2

b. x+2y = -1 diubah menjadi x = -1-2y

sehingga

4(-1-2y)-3y = 7

-4-8y-3y = 7

-11y = 7+4

-11y = 11

[tex] \sf y=- \dfrac {11}{11} [/tex]

y = -1

sehingga x

x+2y = -1

x+2(-1) = -1

x+(-2) = -1

x = -1+2

x = 1

36. Soal matematika transformasi kelas 11 semester 1


Jawaban:

itu jawabannya dah difotoin beserta jalannya, semoga membantu:)

Penjelasan dengan langkah-langkah:

HP = { y - 2x - 3 = 0 }

jalannya tertera di foto

Pencerminan A(x, y)

→ terhadap garis y = -x adalah A'(-y, -x)

→ terhadap garis y = x adalah A'(y, x)

A(x, y) → refleksi y = -x => A'(-y, -x)

A'(-y,-x) → refleksi y = x => A"(-x, -y)

Hingga (x → -x, y → -y)

y = 2x - 3

→ -y = -2x - 3

→ y = 2x + 3 atau 2x - y + 3 = 0

Kesimpulan

Persamaan bayangan garis y = 2x - 3 setelah direfleksikan terhadap garis y = -x, lalu dilanjutkan dengan pengrefleksian terhadap garis y = x adalah y = 2x + 3
