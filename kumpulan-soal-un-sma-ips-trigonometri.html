2. soal dari buku detik-detik UN matematika SMA IPA kategori : trigonometri soal nomor 28


Kelas            : 11
Mapel           : Matematika
Kategori      : Trigonometri lanjut
Kata kunci  : trigonometri, aturan cosinus
Kode            : 11.2.3 (Kelas 11 Matematika Bab 3-Trigonometri lanjut)

Gunakan aturan cos untuk segitiga ABD:
BD²=AB²+AD²-2.AB.AD. cos A

Gunakan aturan cos untuk segitiga BCD:
BD²=BC²+CD²-2.BC.CD. cos C

Berdasarkan sifat segiempat tali busur, sudut A dan C berjumlah 180° (karena berhadapan)

BD²=BD²
AB²+AD²-2.AB.AD. cos A=BD²=BC²+CD²-2.BC.CD. cos C
AB²+AD²-2.AB.AD. cos A=BD²=BC²+CD²-2.BC.CD. cos (180°-A)
AB²+AD²-2.AB.AD. cos A=BD²=BC²+CD²+2.BC.CD. cos A
AB²+AD²-BC²-CD² = 2.BC.CD. cos A+2.AB.AD. cos A
AB²+AD²-BC²-CD² = cos A (2.BC.CD+2.AB.AD)
cos A = (AB²+AD²-BC²-CD²) / (2.BC.CD+2.AB.AD)
cos A = (4²+6²-3²-3²) / (2.3.3+2.4.6)
cos A = (16+36-9-9) / (18+48)
cos A = 34/66
cos A= 17/33

Jadi, cos ∠BAD adalah 17/33

6. rangkuman materi trigonometri tingkat sma di tvri tgl 16​


Rangkuman materi trigonometri tingkat SMA di TVRI tanggal 16 April 2020. Materi trigonometri yang dipelajari adalah materi di matematika wajib kelas X tentang perbandingan trigonometri pada segitiga siku-siku

Pembahasan  

Disajikan segitiga siku-siku ABC, ADE, AFG dan AHI dengan BC // DE // FG // HI (gambar dapat dilihat di lampiran)

ΔABC ≈ ΔADE ≈ ΔAFG ≈ ΔAHI, sehingga berlaku perbandingan

[tex]\frac{BC}{AC} = \frac{DE}{AE} = \frac{FG}{AG} = \frac{HI}{AI} [/tex] = sin A [tex]\frac{AB}{AC} = \frac{AD}{AE} = \frac{AF}{AG} = \frac{AH}{AI} [/tex] = cos A [tex]\frac{BC}{AB} = \frac{DE}{AD} = \frac{FG}{AF} = \frac{HI}{AH} [/tex] = tan A

Perhatikan segitiga ABC siku-siku di B (gambar ada di lampiran)

AB = c = sisi samping (sa) BC = a = sisi depan (de) AC = b = sisi miring

sin A = [tex]\frac{de}{mi} = \frac{BC}{AC} = \frac{a}{b} [/tex]  

cos A = [tex]\frac{sa}{mi} = \frac{AB}{AC} = \frac{c}{b} [/tex]  

tan A = [tex]\frac{de}{sa} = \frac{BC}{AB} = \frac{a}{c} [/tex]  

cosec A = csc A = [tex]\frac{1}{sin \: A} = \frac{AC}{BC} = \frac{b}{a} [/tex]

sec A = [tex]\frac{1}{cos \: A} = \frac{AC}{AB} = \frac{b}{c} [/tex]

cotan A = cot A = [tex]\frac{1}{tan \: A} = \frac{AB}{BC} = \frac{c}{a} [/tex]

Contoh Soal nomor 1

Diketahui segitiga ABC yang siku siku di titik B. AC = 8 cm dan BC = 6 cm. Tentukan perbandingan-perbandingan trigonometri pada segitiga tersebut berdasarkan sudut A

Jawab

Mencari panjang AB

AB = [tex]\sqrt{AC^{2} - BC^{2}}[/tex]

AB = [tex]\sqrt{8^{2} - 6^{2}}[/tex]

AB = [tex]\sqrt{64 - 36}[/tex]

AB = [tex]\sqrt{28}[/tex]

AB = [tex]\sqrt{4 \times 7}[/tex]

AB = [tex]2 \sqrt{7}[/tex]

berarti

sisi depan sudut A = de = BC = 6 sisi samping sudut A = sa = AB = 2√7   sisi miring = mi = AC = 8

Perbandingan trigonometri pada segitiga tersebut adalah

sin A = [tex]\frac{de}{mi} = \frac{BC}{AC} = \frac{6}{8} = \frac{3}{4}[/tex]  

cos A = [tex]\frac{sa}{mi} = \frac{AB}{AC} = \frac{2 \sqrt{7}}{8} = \frac{\sqrt{7}}{4} = \frac{1}{4} \sqrt{7} [/tex]  

tan A = [tex]\frac{de}{sa} = \frac{BC}{AB} = \frac{6}{2 \sqrt{7}} = \frac{3}{\sqrt{7}} = \frac{3}{7} \sqrt{7} [/tex]  

cosec A = [tex]\frac{1}{sin \: A} = \frac{1}{\frac{3}{4}} = \frac{4}{3}[/tex]  

sec A = [tex]\frac{1}{cos \: A} = \frac{1}{\frac{\sqrt{7}}{4}} = \frac{4}{\sqrt{7}} = \frac{4}{7} \sqrt{7} [/tex]      

cotan A = [tex]\frac{1}{tan \: A} = \frac{1}{\frac{3}{\sqrt{7}}} = \frac{\sqrt{7}}{3} = \frac{1}{3} \sqrt{7} [/tex]    

Contoh Soal nomor 2  

Diketahui cos α = [tex]\frac{3}{5}[/tex]. Tentukan nilai sin α dan tan α

Jawab

cos α = [tex]\frac{3}{5}[/tex]

[tex]\frac{sa}{mi} = \frac{3}{5}[/tex]

Sisi samping sudut α = sa = 3 Sisi miring = mi = 5

maka

Sisi depan sudut α

de = [tex]\sqrt{mi^{2} - sa^{2}}[/tex]

de = [tex]\sqrt{5^{2} - 3^{2}}[/tex]

de = [tex]\sqrt{25 - 9}[/tex]

de = [tex]\sqrt{16}[/tex]

de = 4

Jadi nilai sin α dan tan α adalah

sin α = [tex]\frac{de}{mi} = \frac{4}{5}[/tex]  tan α = [tex]\frac{de}{sa} = \frac{4}{3}[/tex]

21. tolong mau dikumpulkan,soal trigonometri​


Jawab:

6,4√2 m

Penjelasan dengan langkah-langkah:

DIKETAHUI:

Tinggi 1 lantai=3,2 m

Sudut kemiringan=45°

DITANYA:

Selisih panjang tangga yang diperlukan untuk lantai 1 dan 3

CARA:

Untuk soal ini, yang diketahui tingginya, yaitu 3,2 m. Kita konversikan ke cm hingga menjadi 320 cm. Maka kita cari sisi miringnya dengan menggunakan sinus.

sin=depan/miring=tinggi 1 lantai/panjang tangga

sin 45=1/√2

         =320/320√2

c=320√2 cm

Kalau untuk 3 lantai, maka tingginya menjadi 960 cm. Kita gunakan rumus sinus sekali lagi.

sin 45=1/√2

         =960/960√2

Maka panjangnya menjadi 960√2 cm.

Sekarang kita cari selisihnya.

960√2 cm - 320√2 cm=640√2 cm

Kita kembalikan menjadi bentuk meternya, yaitu 6,4√2 m.
