Berikut jawabannya dalam bahasa C:

#define _CRT_SECURE_NO_WARNINGS

#include<stdio.h>

void printArray(int cols,int rows,int matrix1[3][2],int matrix2[3][2])

{

printf("Matrix 1:\n");

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  printf("%d ",matrix1[i][j]);

 }

 printf("\n");

}

printf("\nMatrix 2:\n");

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  printf("%d ",matrix2[i][j]);

 }

 printf("\n");

}

}

void swapArray(int cols,int rows,int matrix1[3][2],int matrix2[3][2])

{

int temp;

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  temp = matrix1[i][j];

  matrix1[i][j] = matrix2[i][j];

  matrix2[i][j] = temp;

 }

 

}

}

int main()

{

int cols = 2;

int rows = 3;

int matrix1[3][2] =  

{

 {3,1},

 {4,2},

 {5,3}

};

int matrix2[3][2] =  

{

 {9,4},

 {6,8},

 {7,1}

};

printArray(cols,rows,matrix1,matrix2);

swapArray(cols,rows,matrix1,matrix2);

printf("\nAfter Swap\n\n");

printArray(cols,rows,matrix1,matrix2);

getchar();

return 0;

}

Penjelasan:

Program ini pertama menginisialisasi jumlah baris dan kolom. Kemudian matrix1 dan matrix2 diinisialisasi dengan nilai yang telah ditentukan. Untuk mengetahui perbedaan antara sebelum dan sesudah maka dibuatlah method printArray untuk mencetak matrix1 dan matrix2. Method printArray digunakan 2 kali untuk melihat nilai sebelum dan sesudah. Terakhir ada method untuk swap nilai matrix 1 dan matrix 2 dengan menggunakan bantuan variable temp untuk menyimpan nilai matrix 1 sebelum dioverwrite nilai matrix2 dan digunakan untuk mengoverwrite nilai matrix2.

Jika [tex] R = S^T [/tex] , tentukan nilai [tex] x+y [/tex]

Jawab:

[tex] \begin{align} R &= S^T \\ \begin{pmatrix} 5&2\\4&-3\end{pmatrix} &= \begin{pmatrix} 5&\frac{1}{2}x\\ 4y&-3 \end{pmatrix}^T \\ \begin{pmatrix} 5&2\\4&-3\end{pmatrix} &= \begin{pmatrix} 5&4y\\ \frac{1}{2}x&-3 \end{pmatrix} \end{align} [/tex]

Diperoleh dari kesamaan nilai matriks di atas:

[tex] \begin{align} 2&=4y \\ y &= \dfrac{2}{4} = \dfrac{1}{2} \end{align} [/tex]

kemudian:

[tex] \begin{align} 4 &= \dfrac{1}{2}x \\ x &= 4\cdot 2 = 8 \end{align} [/tex]

Sehingga nilai [tex] x+y = 8 + \frac{1}{2} = 8 \frac{1}{2} = \frac{17}{2} [/tex]

40. Bagaimana ciri-ciri matriks yang memiliki invers matriks? - Brainly #Backtoschool2019


MATRIKS

Suatu matriks persegi atau bujur sangkar memiliki invers jika matriks tersebut non-singular.

Jika matriks A dengan det [A] = 0, maka matriks A dinamakan matriks singular. Setiap matriks singular tidak memiliki invers.Jika matriks A dengan det [A] ≠ 0, maka matriks A dinamakan matriks non-singular. Setiap matriks non singular selalu memiliki invers.

Ciri-ciri matriks memiliki invers:

Matriks yang memiliki ordo sama. Misal: matriks A berordo 2 × 2 dan matriks B berordo 3 × 3.Matriks non-singular atau determinan ≠ 0.

Contoh:

Matriks A (terlampir)

det [A] = ad - bc

det [A] = (1 × 4) - (3 × 2)

det [A] = 4 - 6

det [A] = -2 → {-2 ≠ 0}

Karena det [A] = -2 (≠ 0), maka matriks A merupakan matriks non-singular yang berarti memiliki invers.
