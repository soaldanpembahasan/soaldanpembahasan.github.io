3. di antara pilihan berikut yang merupakan contoh IP address yang benar adalah


Setiap komputer yang tergabung dalam sebuah jaringan akan ditandai dengan alamat yang berbeda-beda. Kalau diibaratkan, IP address ini seperti alamat rumah pada sebuah jalan. Jaringan yang akan kita bangun diistalahkan sebagai jalanan, dan komputer-komputer yang tergabung dalam jaringan tersebut diibaratkan rumah-rumah.

Jadi antara komputer memiliki IP address yang berbeda-beda antara satu dengan yang lainnya. Bayangkan apa jadinya bila dalam sebuah jalan terdapat dua rumah yang memiliki nomor alamat yang sama. Jika terdapat surat yang hendak dikirim ke nomor alamat yang sama tersebut, pastilah surat tersebut tidak akan pernah sampai pada tujuannya, karena bingung mana alamat yang benar. Begitu pula dengan penggunaan IP address pada setiap komputer yang ada dalam jaringan. Jika terdapat alamat yang sama.

Penjelasan:

Jumlah subnet dihitung dengan 2ˣ, x adalah jumlah angka 1 dari oktet terakhir dari subnet masknya, karena subnet mask dgn prefix /26 bernilai 255.255.255.192, angka 192 memiliki binary 11000000 maka 2² = 4

Jumlah Client dapat dihitung dengan 2ⁿ-2, dimana n adalah kebalikan dari x yaitu jumlah 0 pada oktet terakhir yaitu 6, jadi 2⁶-2 = 62

Subnet terdiri dari 192.168.4.0, 192.168.4.64, 192.168.4.128, 192.168.4.192

maka dengan itu IP range (untuk host) dapat diketahui yaitu

192.168.4.1 - 192.168.4.62 dan 192.168.4.63 sbg broadcastnya subnet 192.168.4.0

192.168.4.65 - 192.168.4.126 dan 192.168.4.127 sbg broadcastnya subnet 192.168.4.64

192.168.4.129 - 192.168.4.190 dan 192.168.4.191 sbg broadcastnya subnet 192.168.4.128

15. jika Ip 192.168.18.177 brp ip subnet dan ip broad cast


Jika IP tersebut memiliki Mask 255.255.255.0 / Class C Full
maka IP Subnet adalah 192.168.18.0 dan IP Broadcast 192.168.18.255Cara menghitung IP Subnet (IP Network):
Ubah IP menjadi biner:
11000000.10101000.00010010.10110001
Tidak ada suffix (Nilai CIDR)? Gunakan default netmask: /24 yaitu 255.255.255.0.
Ubah Subnet mask tsb menjadi biner juga:
11111111.11111111.11111111.00000000

Perhitungan Network:
Subnet mask AND IP:
[tex]11111111.11111111.11111111.00000000[/tex]
[tex]11000000.10101000.00010010.10110001[/tex]
-------------------------------------------------------- AND
[tex]11000000.10101000.00010010.00000000[/tex] ATAU 192.168.18.0

Perhitungan Broadcast:
Invers Subnet Mask OR Network
[tex]00000000.00000000.00000000.11111111[/tex]
[tex]11000000.10101000.00010010.00000000[/tex]
------------------------------------------------------------------- OR
[tex]11000000.10101000.00010010.11111111[/tex] ATAU 192.168.18.255

Maka dengan IP 192.168.18.177, 
subnet masknya adalah 255.255.255.0

Networknya adalah 192.168.18.0 dengan broadcast 192.168.18.255

34. 192.196.3.121/23 carilah IP subnet? carilah ip network? carilah ip awal? carilah ip akhir? carilah ip broadcash?


192.196.3.121 / 23

Netmask : 11111111.11111111.11111110.00000000 ----> 255.255.254.0
Jumlah subnet : 2⁷ = 128
Jumlah host / subnet = 2⁹  - 2 = 512 - 2  = 510
Blok subnet = 256 - 254 = 2
-----> 0,2,4,6,8,.....,254

IP Network  = IP Subnet

IP Network                      IP pertama                  IP terakhir                Broadcast
192.196.0.0                    192.196.0.1               192.196.1.254            192.196.1.255
192.196.2.0                    192.196.2.1               192.196.3.254            192.196.3.255
192.196.4.0                    192.196.4.1               192.196.5.254            192.196.5.255
.................                         .................                ...................            ......................
192.196.254.0                 192.196.254.1           192.196.255.254        192.196.255.255

Jadi IP 192.168.3.121 terletak di subnet ke-2 dengan :
IP Subnet / IP Network = 192.196.2.0
IP Pertama = 192.196.2.1
IP Terakhir = 192.196.3.254
IP Broadcast = 192.168.3.255
