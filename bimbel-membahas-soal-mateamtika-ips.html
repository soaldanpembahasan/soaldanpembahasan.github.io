IPA = 75
IPA saja = 75 - x
IPS = 65
IPS saja = 65 - x
Kedua duanya (IPA & IPS) = x
Tidak keduanya = 40
Jumlah = 150 orang

=> (75 - x) + (65 - x) + x + 40 = 150
=> 180 - x = 150
=> - x = 150 - 180
=> - x = - 30
=> x = 30

=>Jadi banyak orang yang mengikuti bimbel IPA & IPS = 30 orang
