7. Buatlah sebuah program dengan bahasa c untuk menentukan predikat akademik di akhir semester berdasarkan nilai IP Semester (IPS) yang ada dalam skala 0-4, dan predikat di semester sebelumnya. Aturan akademik perguruan tinggi tersebut mengenai predikat semester adalah: a. Jika predikat sebelumnya adalah PERINGATAN, maka predikat semester ini adalah: i. DIKELUARKAN jika IPS kurang dari 2.0 ii. TANPA SYARAT jika IPS sedikitnya 2.0 b. Jika predikat sebelumnya adalah TANPA SYARAT, maka predikat semester ini adalah: i. DIKELUARKAN jika IPS kurang atau sama dengan 1.5 ii. PERINGATAN jika IPS lebih dari 1.5 tetapi kurang dari 2.0 iii. TANPA SYARAT jika IPS sedikitnya 2.0


#include <stdio.h>

int main() {

 int i, j, nilai, jmlsmt, jmlmk, sks[50][30], jumlahnilai, skssmt[14], jumlahsks, totalsks=0;

 char mk[30], nilaihuruf[50][30], matkul[50][50][30];

 float ipk,nr[14], totalnr=0;

 printf("==============================================\n");

 printf("\tProgram Menghitung IPK Mahasiswa\n");

 printf("==============================================\n");

 printf("Masukkan jumlah semester: ");

 scanf("%d", &jmlsmt);

 if (jmlsmt < 2 || jmlsmt > 14) {

   printf("Jumlah semester salah!\n");

   return 0;

 }

 else {

   for (i = 0; i < jmlsmt; i++) {

     jumlahnilai = 0;

     jumlahsks = 0;

     printf("Masukkan jumlah mata kuliah semester %d: ", i + 1);

     scanf("%d", &jmlmk);

     if (jmlmk < 2) {

       printf("Jumlah matakuliah kurang dari 2 setiap semester\n");

       return 0;

     }

     else {

       for (j = 0; j < jmlmk; j++) {

         printf("Masukkan mata kuliah ke %d\n", j + 1);

         printf("Masukkan nama matkul: ");

         scanf(" %s", matkul[i][j]);

         printf("Masukkan jumlah sks matkul: ");

         scanf("%d", &sks[i][j]);

         printf("Masukkan nilai matkul: ");

         scanf(" %c", &nilaihuruf[i][j]);

         printf("--------------------------------------------\n");

         if (nilaihuruf[i][j] == 'A') {

           nilai = 4 * sks[i][j];

         }

         else if (nilaihuruf[i][j] == 'B') {

           nilai = 3 * sks[i][j];

         }

         else if (nilaihuruf[i][j] == 'C') {

           nilai = 2 * sks[i][j];

         }

         else if (nilaihuruf[i][j]=='D') {

           nilai = 1 * sks[i][j];

         }

         else if (nilaihuruf[i][j]=='E') {

           nilai = 0 * sks[i][j];

         }

         else {

           printf("Input salah!\n");

           return 0;

         }

         jumlahnilai = jumlahnilai + nilai;

         jumlahsks = jumlahsks + sks[i][j];

       }

       if(jumlahsks > 24){

         printf("Jumlah SKS semester lebih dari 24\n");

         return 0;

       } else {

         skssmt[i] = jumlahsks;

         nr[i] = jumlahnilai / jumlahsks;

       }    

     }

   }

 }

 printf("==============================================\n");

 printf("\t\tTranskrip Nilai\n");

 printf("==============================================\n");

 for(i = 0; i < jmlsmt; i++){

   printf("\nHasil Semester %d :\n", i+1);

   printf("\n%12s%12s%12s","Mata Kuliah", "SKS", "Nilai\n");

   for(j = 0; j < jmlmk; j++){

     printf("%12s%12d%12c\n",matkul[i][j],sks[i][j],nilaihuruf[i][j]);

   }

   printf("\nSKS\t: %d\n", skssmt[i]);

   printf("NR\t: %.2f\n", nr[i]);

   totalsks = totalsks + skssmt[i];

   totalnr = totalnr + nr[i];

   printf("--------------------------------------------\n");

 }

 ipk = totalnr/jmlsmt;

 printf("\nTotal SKS\t: %d\n", totalsks);

 printf("IPK\t\t: %.2f\n", ipk);

 printf("==============================================\n");

 return 0;

}

10. perubahan IPS dari orientasi pada disiplin akademik yang terpisah-pisah ke suatu upaya untuk mencari hubungan interdisipline terdapat pade era​


Jawaban:

Perubahan dalam pendekatan Ilmu Pengetahuan Sosial (IPS) dari orientasi pada disiplin akademik yang terpisah-pisah menuju upaya mencari hubungan interdisipliner dapat diamati dalam era saat ini.

Tradisionalnya, IPS terdiri dari disiplin akademik yang terpisah seperti sejarah, geografi, ekonomi, sosiologi, politik, dan antropologi. Setiap disiplin ini memiliki fokus dan metodologi sendiri untuk mempelajari aspek tertentu dari masyarakat dan manusia. Namun, dalam beberapa tahun terakhir, ada pergeseran paradigma menuju pendekatan interdisipliner dalam IPS.

Pendekatan interdisipliner dalam IPS mencoba untuk melihat fenomena sosial secara holistik dan menyeluruh, mengintegrasikan berbagai disiplin dan perspektif untuk memahami kompleksitas masyarakat dan hubungan antarbagian yang saling mempengaruhi. Ini mengakui bahwa fenomena sosial tidak dapat dijelaskan secara memadai hanya melalui satu disiplin tunggal.

Beberapa alasan mengapa perubahan ini terjadi di era saat ini adalah:

1. Kompleksitas masalah global: Masalah sosial saat ini semakin kompleks dan saling terkait, seperti perubahan iklim, globalisasi, migrasi, dan ketimpangan ekonomi. Menggunakan pendekatan interdisipliner memungkinkan kita untuk mendapatkan pemahaman yang lebih lengkap tentang fenomena ini dan menemukan solusi yang lebih efektif.

2. Pemahaman yang lebih mendalam: Pendekatan interdisipliner memungkinkan kita untuk memperluas wawasan dan perspektif kita dengan memanfaatkan pemikiran dan metodologi dari berbagai disiplin. Ini dapat menghasilkan pemahaman yang lebih dalam tentang fenomena sosial yang kompleks.

3. Kolaborasi lintas disiplin: Dalam era digital dan konektivitas global, kolaborasi lintas disiplin menjadi lebih mudah. Peneliti, akademisi, dan praktisi dapat bekerja bersama untuk mengintegrasikan pengetahuan dan perspektif mereka, menghasilkan pemahaman yang lebih baik dan solusi yang inovatif.

4. Permintaan pasar tenaga kerja: Pasar tenaga kerja saat ini membutuhkan individu dengan keterampilan interdisipliner. Kemampuan untuk mengintegrasikan dan menerapkan pengetahuan dari berbagai disiplin menjadi lebih berharga dalam memecahkan masalah kompleks dan beradaptasi dengan perubahan yang cepat.

Dalam kesimpulannya, perubahan dalam IPS dari orientasi pada disiplin akademik yang terpisah-pisah menjadi upaya untuk mencari hubungan interdisipliner adalah respons terhadap kompleksitas fenomena sosial dan permintaan pemahaman yang lebih mendalam. Pendekatan interdisipliner memungkinkan kita untuk melihat fenomena sosial secara holistik dan menggabungkan berbagai disiplin untuk memperoleh wawasan yang lebih kaya dan solusi yang lebih baik.

17. pengertian pretest adalah​


Pengertian pre-test adalah suatu pengujian yang dilakukan pada sekelompok variabel tertentu sebelum diberikan perlakuan (penyampaian materi atau tindakan tertentu lain).

Pembahasan

Indikator untuk mengetahui kemampuan dan/atau pemahaman dari "responden" (target penelitian) mengenai suatu hal (materi tertentu) adalah pre-test dan post-test. Pre-test merupakan serangkaian tes atau hanya tes tunggal untuk mengetahui kemampuan dari responden sebelum materi diberikan. Sedangkan, post-test adalah tes yang diberikan setelah responden memperoleh/menerima materi tertentu terkait tes.

Contoh penelitian metode pembelajaran perkalian dan pembagian bilangan bulat dari 1-10 pada siswa kelas 3 SD. Maka pre-test dapat dengan menggunakan aplikasi Kahoot untuk dapat mengukur kualitas siswa itu sebelum ikut pembelajaran. Setelah diberikan pembelajaran, siswa kelas 3 diberikan tes lagi dengan aplikasi yang sama dan soal yang miri[. Hasil dari post-test kemudian dibandingkan dengan hasil pre-test untuk mengetahui sebera baik metode pembelajaran yang diberikan.

Jawaban:

Pre test dan post test adalah adalah bentuk evaluasi pembelajaran yang dilakukan guru kepada peserta didik.

Kedua bentuk evaluasi ini sering digunakan untuk mengukur kompetensi awal dan kompetensi akhir mereka.

Kompetensi awal merupakan tingkat pemahaman peserta didik sebelum menerima pembelajaran, sedangkan kompetensi akhir merupakan tingkat penguasaan materi peserta didik setelah menerima pembelajaran.

Tingkat pemahaman peserta didik terhadap materi yang dipersyaratkan dan tujuan pembelajaran tertentu perlu untuk dievaluasi menggunakan pre test dan post test.

1. Pre Test

Secara sederhana, pengertian pre test adalah tes yang dilakukan sebelum guru memulai pembelajaran.

Fumgsi pre test adalah untuk mengetahui kemampuan awal peserta didik terkait materi yang akan disampaikan.

Dengan mengetahui kemampuan awal tersebut, maka guru lebih mudah untuk menentukan model dan metode yang akan diterapkan dalam pembelajaran.
