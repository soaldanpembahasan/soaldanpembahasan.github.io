15. Rata-rata nilai PAT matematika kelas 8A adalah 82. Rata-rata nilai PAT siswa laki-laki di kelas 8A adalah 80 dan rata-rata nilai PAT siswa perempuan di kelas 8A adalah 86. Jika banyak siswa kelas 8A ada 30 anak, tentukan banyak siswa perempuan di kelas 8A tersebut!


Jawaban:

Jumlah siswa kelas 8A = a

Jumlah siswa kelas 8B = b → 15 siswa

Nilai rata - rata a () = 72

Nilai rata - rata b () = 80

Nilai rata - rata gabungan () = 75

secara umum, menentukan nilai suatu rata-rata didefenisikan sebagai hasil dari jumlah keseluruhan nilai data dibagi jumlah banyak siswa atau frekuensi data.

Langkah Pertama → Hitung Jumlah nilai siswa kelas 8b

Jumlah nilai b = banyak siswa 8B x nilai rata-rata

= b x 80

= 80b

= 80 x 15

= 1.200

Jumlah nilai a = banyak siswa 8A x nilai rata - rata

= a x 72

= 72a

Maka, menentukan banyaknya siswa kelas 8A dapat dihitung dengan cara sebagai berikut :

→ Lakukan perkalian silang

72a + 1.200 = 75 (a + 15)

72a + 1.200 = 75a + (75 x 15)

72a + 1.200 = 75a + 1.125 → Gabungkan variabel a

1.200 - 1.125 = 75a - 72a

75 = 3a

= a

25 = a → Banyak siswa kelas 8A

Jadi, Banyak siswa kelas 8A adalah 25 siswa

Luas Taman Tomat
