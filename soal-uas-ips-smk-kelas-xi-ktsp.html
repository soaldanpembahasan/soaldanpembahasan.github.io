9. 5. Tinggi rata-rata dari 50 siswa adalah 164,6 cm, jika Ibu Guru Matematika Kelas XI IPS diikutkan, maka tinggi rata-rata 164,64 cm. Maka tinggi Ibu Guru Matematika Kelas XI SMK tersebut adalah ... cm.​


diket :

rata-rata 1 ≈ 164,6 cm (50 siswa)

rata-rata 2 ≈ 164,64 cm (50 siswa + 1 guru)

ditanya :

tinggi ibu guru = x

jawab :

1. rata-rata 1 × 50 siswa

= 164,6 × 50 siswa

= 8230 cm

2. rata-rata 2 × 51 siswa + guru

= 164,64 × 51 siswa dan ibu guru

= 8396,64 cm

× =jumlah tinggi 51 siswa danguru-jumlah tinggi 50siswa

= 8396,64 - 8230

= 166,64 cm

jadi, tinggi ibu guru adalah 166,64 cm

semoga bermanfaat :)

a. Persamaan lingkaran berpusat (-2,5), melalui (-1,2)

(x + 2)² + (y - 5)² = r²

→ (-1 + 2)² + (2 - 5)² = 10

→ x² + y² + 4x - 10y + 4 + 25 - 10 = 0

→ x² + y² + 4x - 10y + 19 = 0

b. Persamaan lingkaran berpusat (-2,5) menyinggung garis x - 3 = 0 → x = 3

(x + 2)² + (y - 5)² = r²

→ (3 + 2)² + (0 - 5)² = r²

→ r² = 25

→ (x + 2)² + (y - 5)² = 25

→ x² + y² + 4x - 10y + 4 + 25 - 25 = 0

→ x² + y² + 4x - 10y + 4 = 0
