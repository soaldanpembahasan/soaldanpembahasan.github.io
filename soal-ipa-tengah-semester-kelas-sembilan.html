26. Perhatikan gambar berikut!

Jika besar 2 CDO = 25° dan CBO = 329, maka besar Z AOD adalah.....

A. 57°. B. 66°. C. 114. D. 123

Cara Jawaban =

24.)

Panjang = ( 6 x Diameter ) + K. Lingkaran.

Panjang = ( 6 x 20 cm ) + ( 22/7 x 20 cm. )

Panjang = 120 cm + 62,8 = 182,8 cm ( B. )

25.)

29 x 2 = 58 x 2 = 116.

20 x 2 = 40 x 2 = 80.

116 + 80 = 196.

30.000 x 196 = 558.000.

2 x ( 29 x 2 ) + 2 ( 16 x 2 )

(2 x 58) + (2 x 32) = 116 + 64 = 180.

180 x 30.000 = 5.400.000. ( C. )

26.)

∠AOD = 180⁰ - ( ∠CDO + ∠CBO ) x 2.

∠AOD = 180⁰ - ( 25⁰ + 32⁰ ) x 2.

∠AOD = 180⁰ - ( 57⁰ x 2 ).

∠AOD = 180⁰ - 114⁰ = 66⁰ ( B. )

Kesimpulan Jawaban =

24.) B. 182,8 cm.

25.) C. 5.400.000.

26.) B. 66⁰.
