D . {2, 4}

2.Jika M = {a, i, u, e, o} , N = {a, u, o}, nilai n dari (M N) yaitu ….

A . 5

B . 6

C . 7

D . 8

3.Jika telah terlihat X = {x | x <6, x bilangan asli} sedangkan nilai Y = {x | – 1 x 5, x integer}, maka anggota (X Y) adalah ….

A . {0, 1, 2, 3, 4, 5}

B . {1, 2, 3, 4, 5}

C . {-1, 0, 1, 2, 3, 4}

D . {-1, 0, 1, 2, 3, 4, 5}

4 . nilai n (A) = 10, nilai n (B) = 8 ada n (A B) = 8, maka nilai n atas (A B) adalah ….

A . 8

B . 9

C . 10

D . 11

5.belakang nilai C = {bilangan asli <10} sedangkan nilai A = {2, 4, 6, 8}. Maka nilai dari AC adalah….

A . {1, 2, 3, …., 9}

B . {0, 1, 3, 5, 7, 9}

C . {2, 4, 6, 8}

D . {1, 3, 5, 7, 9}

jawabannya dan pembahasannya:

1.Pembahasan

Jika:

A = {5,6,7}

B = {3,4}

Maka nilai dari A B = {3,4,5,6,7}

Jawabannya: B

2.Pembahasan

Jika:

M = {a, i, u, e, o}

N = {a, u, o}

M N = {a, i, u, e, o}

Nilai n (M N) = 5

Jawabannya : A

3.Pembahasan

Jika

X = {x | x <6, x bilangan angka asli)

= {1, 2, 3, 4, 5}

Y = {x | – 1 x 5, x nilai integer}

= {-1, 0, 1, 2, 3, 4, 5}

(X Y) = {1, 2, 3, 4, 5}

Jawabannya: B

4.Pembahasan

n (A) = 10, n (B) = 8 serta n (A B) = 8

n (A – B) = n (A) + n (B) -n (AB)

= 10 + 8 – 8

= 10

Jawabannya: C

5.Pembahasan

S = {1, 2, 3, 4, 5, 6, 7, 8, 9}

A = {2, 4, 6, 8}

Ac = {1, 3, 5, 7, 9}

Jawabannya : D
