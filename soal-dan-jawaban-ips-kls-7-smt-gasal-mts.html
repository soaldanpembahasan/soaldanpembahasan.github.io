Penjelasan:

Kata ganti berfungsi untuk menggantikan kata benda yang sebelumnya telah disebutkan atau telah diketahui sehingga membuat kalimat lebih efisien dan tidak boros kata, kata ganti dalam Grammar bahasa Inggris disebut sebagai possessive pronoun, namun dalam Nahwu bahasa Arab disebut dengan dhomir, kata ganti untuk kepemilikan seperti yang telah dijelaskan pada kolom jawaban di atas dapat diaplikasikan ke dalam contoh kata seperti di bawah ini;

كِتَابي = bukukuكِتَابينا = buku kitaكِتَابكَ = bukumu (laki laki)كِتَابكِ = bukumu (perempuan)كِتَابكُما = bukumu (dua orang)كِتَابكُم = buku kalian (laki laki)كِتَابكُنَّ = buku kalian (perempuan)كِتَابهُ = bukunya (laki laki)كِتَابهُما = buku mereka (berdua)كِتَابهُم = buku mereka (laki laki)كِتَابها = bukunya (perempuan) كِتَابهُنَّ = buku mereka (perempuan)

17. tolong ajarin no 7...kelas X smt gasalmakasih


|4x - 3| = |x - 7|
|4x - 3| - |x - 7| = 0
(4x - 3)² - (x - 7)² = 0
(4x - 3 + (x - 7))(4x - 3 - (x - 7)) = 0
(4x - 3 + x - 7)(4x - 3 - x + 7) = 0
(5x - 10)(3x + 4) = 0

5x - 10 = 0
5x = 10
x = 2

dan

3x + 4 = 0
3x = -4
x = -4/3
