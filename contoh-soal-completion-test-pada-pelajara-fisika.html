20. kak kalau test akademik smp itu pelajaranya dari kelas 6 y kak


Rata-rata semua tidak ada di pelajaran kelas 6, cuma kasih info
TPA (Tes Potensi Akademik) itu dibagi 3
1.) Verbal
2.) Numerik
3.) Figural

kalau verbal, itu soalnya seperti nyamain sesuatu, misal :
Gendut : kurus = Jahat : ....
Jawabannya adalah Baik, karena itu adalah antonim
Ini aku kasih soal, kamu latihan sendiri yah ;)
1.) Mata : air mata = Hidung : ....
2.) Buku : pintar = Game : ...
3.) Apel : merah = Pisang : ....

kalau numerik biasanya itung-itung, kebanyakan yang muncul adalah pola bilangan
contoh soal kayak gini : 
2,4,6,8,..,.. 
nah disitu kamu disuruh nyebutin pola selanjutnya
caranya, lihat polanya disitu kan polanya +2,+2,+2,+2 dan seterusnya, jadi selanjutnya juga ditambah 2
jawabannya 10,12

ini coba kamu kerjain ;)
1.) 3,6,9,12,..
2.) 4,8,12,16,...
3.) 2,4,8,16,...

Kalau figural, itu seperti soal tebakan gambar, jadi kamu harus banyak-banyak berlatih ya ;)

5. Persamaan kuadrat yang akar-akarnya 3 dan -2 adalah

Penyelesaian

1. [tex]\frac{x^{2} - 2x - 15 }{x^2+ 10x + 21 } \\= \frac{(x + 3)(x -5)}{(x + 7)(x + 3)} \\= \frac{(x - 5)}{(x + 7)}[/tex]

2. [tex]\frac{3 - 5x}{6} < x + 1\\= 3 - 5x < 6(x + 1)\\= 3 - 5x < 6x + 6\\-5x - 6x < 6 - 3\\-11x < 3\\x > -\frac{3}{11}[/tex]

= 17,5 x 22,5

= 393,75 m²

4. S = n(M) + n(BI) + [tex]n(A U B)^{C}[/tex] - x

48 = 58 - x

x = 58 - 48

x = 10 siswa

5. x1 = 3

y = (x - 3)(x - (-2))

y = (x - 3)(x + 2)

y = x² - x - 6

39. (SOAL PELUANG) Pada tes fisika terdapat 5 soal pilihan ganda dan 3 soal essai. Peserta test harus mengerjakan 3 soal pilihan ganda dan  2 soal essai. Banyaknya pilihan untuk mengerjakan test adalah.... cara


Pilihan ganda ada 5 dikerjakan 3 maka ada 5C3 = 10 cara
Esai ada 3 dikerjakan 2 maka ada 3C2 = 3 cara
Jadi, total cara pengerjaan ada 10 . 3 = 30 caradik : 5 soal pilihan ganda
       3 soal essai
      harus mengerjakan 3 soal pilihan ganda dan 2 soal esai
dit : banyak cara untuk mengerjakan test adalah ..?

menggunakan metode kombinasi :
Pilihan berganda : C(5,3) = 5 ! / (5-3) ! . 3 ! = 5.4.3.2! / 2! 3! = 60 / 3.2.1 = 10 cara
Essai : C(3,2) = 3! / (3-2) ! 2! = 3 . 2! / 1 ! 2! = 3/1 = 3 cara

sehingga total cara = 10 + 3 = 13 cara
