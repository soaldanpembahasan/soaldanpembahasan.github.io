Jawaban:

Q1: What is a computer?

A1: A computer is an electronic device that can store, process, and output data.

Q2: What is the purpose of a computer?

A2: The primary purpose of a computer is to accept input, process it, and produce output.

Q3: What are the components of a computer?

A3: The main components of a computer are the processor, memory, storage, and input/output devices.

Q4: What is the difference between a laptop and a desktop computer?

A4: A laptop is a portable computer that runs on a battery, while a desktop computer is a larger stationary computer that is powered by a plug.

Q5: What is a CPU?

A5: A CPU (Central Processing Unit) is the main processing unit of a computer. It is responsible for executing instructions and processing data.

Q6: What is RAM?

A6: RAM (Random Access Memory) is a type of memory used in computers to store temporary data. It is volatile memory, meaning that its contents are erased when the computer is turned off.

Q7: What is a hard drive?

A7: A hard drive is a type of non-volatile storage device that stores data permanently.

Q8: What is an operating system?

A8: An operating system is a set of software that manages the resources of a computer and provides an interface for users to interact with the computer.

Q9: What is an application?

A9: An application is a type of software that performs specific tasks, such as word processing, web browsing, and gaming.

Q10: What is a network?

A10: A network is a group of computers that are connected together to share resources and information.

Q11: How does a computer connect to the internet?

A11: Computers can connect to the internet via a modem or router.

Q12: What is a firewall?

A12: A firewall is a security system that protects a computer from malicious programs and hackers.

Q13: What is a virus?

A13: A virus is a malicious program that can infect a computer and cause damage or slow down its performance.

Q14: What is malware?

A14: Malware is a type of malicious software that can damage, delete, or modify data on a computer.

Q15: What is an antivirus?

A15: An antivirus is a type of software that scans a computer for viruses and other malicious programs and removes them.

Q16: What is a graphics card?

A16: A graphics card is a type of hardware that processes and renders images for display on a computer monitor.

Q17: What is a USB port?

A17: A USB (Universal Serial Bus) port is a type of connection port that is used to connect peripheral devices to a computer.

Q18: What is a printer?

A18: A printer is a type of hardware that can output hard copies of documents from a computer.

Q19: What is a scanner?

A19: A scanner is a type of hardware that can scan documents and convert them into digital files.

Q20: What is a modem?

A20: A modem is a type of hardware that connects a computer to the internet.

1. Folder dan File:Folder: Sebuah folder adalah sebuah wadah yang digunakan untuk mengorganisasi dan menyimpan file-file di dalamnya. Dalam konteks komputer, folder sering kali digunakan untuk mengelompokkan file berdasarkan topik, jenis, atau kategori tertentu. Folder juga dapat berisi subfolder yang lebih spesifik.File: Sebuah file adalah kumpulan data yang disimpan dalam sebuah wadah dengan nama tertentu. File dapat berisi teks, gambar, audio, video, atau jenis data lainnya. File memiliki ekstensi yang menunjukkan jenis atau format file tersebut.

2. Fungsi Folder di Komputer:Organisasi: Folder memungkinkan kita mengatur file-file ke dalam struktur yang terorganisir. Dengan menggunakan folder, kita dapat mengelompokkan file berdasarkan topik, proyek, jenis, atau kategori lainnya, sehingga memudahkan dalam mencari dan mengakses file yang diperlukan.Pemisahan: Folder memungkinkan kita memisahkan file-fIle yang berbeda agar tidak bercampur menjadi satu. Misalnya, kita dapat memiliki folder terpisah untuk dokumen pekerjaan, gambar pribadi, musik, atau video.Pengamanan: Folder dapat digunakan untuk mengamankan file-file tertentu dengan memberikan akses yang terbatas. Dengan mengatur izin akses pada folder, kita dapat melindungi file-file yang sensitif atau pribadi dari akses yang tidak sah.

3. Cara Membuat Folder Baru di Komputer:

Untuk membuat folder baru di komputer, ikuti langkah-langkah berikut:

Buka File Explorer (Windows) atau Finder (Mac).Pilih lokasi di mana Anda ingin membuat folder baru, misalnya di Desktop, di dalam folder tertentu, atau di drive penyimpanan.Klik kanan pada area kosong di lokasi tersebut dan pilih "New" (Windows) atau "File" (Mac), lalu pilih "Folder".Beri nama pada folder baru tersebut dan tekan Enter untuk membuatnya.

4. Ekstensi File dan Fungsinya:

- Ekstensi file adalah bagian dari nama file setelah titik (contoh: .txt, .jpg, .mp3). Ekstensi file menunjukkan jenis atau format file tersebut. Berikut adalah beberapa contoh ekstensi file dan fungsinya:
