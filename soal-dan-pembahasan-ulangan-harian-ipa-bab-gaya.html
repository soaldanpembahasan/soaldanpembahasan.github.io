15. soal tentang IPA bab tekanan


Tekanan

P = F/A
P = (m x g)/(p x l)
P = (40 x 10)/(1 x 0,4)
P = 400/0,4
P = 1000 N/m^2
Jadi besar tekanan kardus terhadap lantai adalah 1000 N/m^2.

Keterangan:
P = tekanan (satuan N/m^2)
F = gaya (satuan N)
A = luas bidang tekan (satuang m^2)
m = massa (satuan kg)
g = percepatan gravitasi (satuan m/s^2)
p = panjang (satuan m)
l = lebar (satuan m)

19. 2ab + 3ab +7ab² - 9ab + 6c​


Pertanyaan:

Hasil dari 2ab + 3ab +7ab² - 9ab + 6c​

Jawab dan Penjelasan dengan langkah-langkah:

2ab + 3ab +7ab² - 9ab + 6c​

= (2ab + 3ab + (-9ab)) + 7ab² + 6c

= (5ab - 9 ab) + 7ab² + 6c

= (-4ab) + 7 ab² + 6c
