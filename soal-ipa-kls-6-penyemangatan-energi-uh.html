40. bantu menjawab soal ini.. saya but uh sekali


Diketahui :
To = 5 K
Tt = 65 K
Ao = 4,0144 m²
Ditanyakan : At ?

Dijawab :
Kofisien muai tembaga :
α = 0,000017
ΔT = 65 - 5 = 60 K = 60 + 273 = 333 C
 
Jadi :
At = Ao (1 + 2α x ΔT)
    = 4,0144 (1 + 2 X0,000017 x 333)
    = 4,0144 (1 + 0,000034 x 333)
    = 4,0144 ( 1+ 0,011322)
    = 4,059851 m²
