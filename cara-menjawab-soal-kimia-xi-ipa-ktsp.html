A. 10 + log 2

Penjelasan:

Be(OH)2 => Be^2+ + 2 OH^-

s s 2s

Ksp = [Be^2+] [OH^-]^2

4x10^-12 = ( s ) ( 2s )^2

4x10^-12 = 4s^3

s = 10^-4 M

[OH-] = 2s => 2x10^-4 M

pOH = -log 2x10^-4

=> 4 - log 2

pH = 14 - ( 4 - log 2 )

=> 10 + log 2

Jawaban:

a. 10 + log 2

Caranya dibawah ini:

Be(OH)2 → Be + 2OH

S 2s

Ksp = [Be+] [OH-]²

4 x 10^12 = (s) (2s)²

4 x 10^12 = (s) (4s²)

4 x 10^12 4.s³

s³ = (4 x 10^12) / 4

s³ = 10^12

s = √(10^12)

s = 10^4 M

[OH-] = 2 x s

= 2 x 10^4

POH = -log [OH-]

= -log (2 x 10^4)

= 4 - log 2

PH = 14-(4- log 2)

= 14 - 4 + log 2 = 10 + log 2

   =  (ΔH CO2 + 2 ΔH H2O ) -  ( ΔH CH3OH + 2 ΔH O2  )  

   =  ( - 393,5 + 2 x - 286 ) kj/mol - ( - 238,6 + 0 ) kj/mol

   =  (( - 393,5 - 572 )  + 238,6) kj/mol

   =   - 965,5 + 238,6 kj/mol

   = - 726,9 kj/mol

  Mr CH3OH = 32

  mol = gram metanol / Mr metanol = 10 gram / 32 = 0,3125 mol

  maka kalor yang dibebaskan = 0,3125 mol x 726,9 kj/mol = 227,16 kj.

25. kimia kelas XI jawab menggunakan cara ​


aturan dalam ion senama:

Semakin banyak ion senama maka semakin kecil kelarutan suatu zat. Begitu juga sebaliknya.

[tex]BaSO_4→{Ba}^{2+}+{SO_4}^{2-}[/tex]

menghasilkan ion Ba2+ dan SO4 2-

Karena di setiap opsi molaritasnya sama, maka kita langsung cek saja jumlah ion senamanya

Air Murni

[tex]H_2O[/tex]

Tidak ada ion senama. Jadi, salah deh.

Barium Klorida

[tex]BaCl_2[/tex]

ion senamanya adalah Ba2+

[tex]BaCl_2→{Ba}^{2+}+2Cl{}^{-}[/tex]

Jumlah ion Ba2+ adalah 1

Asam Sulfat

[tex]H_2SO_4[/tex]

ion senamanya adalah SO4 2-

[tex]H_2SO_4→2{H}^{+}+{SO_4}^{2-}[/tex]

Jumlah ion SO4 2- adalah 1

Alumunium Sulfat

[tex]Al_2(SO_4)_3[/tex]

ion senamanya adalah SO4 2-

[tex]Al_2(SO_4)_3→2{Al}^{3+}+3{SO_4}^{2-}[/tex]

Jumlah ion SO4 2- adalah 3

Amonium Sulfat

[tex](NH_4)_2SO_4[/tex]

ion senamanya adalah SO4 2-

[tex](NH_4)_2SO_4→2{NH_4}^{+}+{SO_4}^{2-}[/tex]

Jumlah ion SO4 2- adalah 1

Jadi, sesuai aturan ion senama di awal, kelarutan BaSO4 paling kecil terdapat dalam Al2(SO4)3
