17. Tentukan net id dan host id dari 10. 10. 10. 1/29 Tentukan total IP Network IP.,broadcast IP jumlah lp host range IP dan range IP host dari IP diatas​


Jawaban:

Mari kita tentukan net ID dan host ID dari alamat IP 10.10.10.1/29.

1. **Net ID:**

- Untuk menentukan net ID, kita harus menetapkan bit host ke nilai nol pada oktet terakhir.

- Alamat IP: 10.10.10.1 (dalam representasi biner: 00001010.00001010.00001010.0000**0001**)

- Net ID: 10.10.10.0 (dalam representasi biner: 00001010.00001010.00001010.0000**0000**)

2. **Host ID:**

- Untuk menentukan host ID, kita harus mempertahankan nilai bit host pada oktet terakhir.

- Host ID: 0.0.0.1 (dalam representasi biner: 0000**0000**.0000**0000**.0000**0000**.0000**0001**)

3. **Total IP:**

- Subnet /29 memiliki total 8 alamat IP.

4. **Network IP:**

- Network IP: 10.10.10.0

5. **Broadcast IP:**

- Broadcast IP: 10.10.10.7

6. **Jumlah IP Host:**

- Jumlah IP host yang dapat digunakan: 8 - 2 = 6 (karena ada dua alamat yang tidak dapat digunakan, yaitu network dan broadcast)

7. **Range IP Host:**

- Range IP host: 10.10.10.1 hingga 10.10.10.6 (alamat 10.10.10.0 adalah network, dan 10.10.10.7 adalah broadcast)

8. **Range IP:**

- Range IP: 10.10.10.0 hingga 10.10.10.7

Jadi, untuk alamat IP 10.10.10.1/29:

- Net ID: 10.10.10.0

- Host ID: 0.0.0.1

- Total IP: 8

- Network IP: 10.10.10.0

- Broadcast IP: 10.10.10.7

- Jumlah IP Host: 6

- Range IP Host: 10.10.10.1 hingga 10.10.10.6

- Range IP: 10.10.10.0 hingga 10.10.10.7

36. Network id, broadcast id, range ip, jumlah host dari ip 192.168.45.1/24


IP address : 192.168.45.1/24 atau

= 192.168.45.1 255.255.255.0

Cara menghitung :

Ubah IP Address dan Netmask kedalam bentuk biner, 192.168.45.1 IP Address tersebut berada di kelas C,  ambil octet ketiga dan ke empat untuk dihitung, maka :

IP address : x.x.0010 1101. 0000 0001 = 45.1

Netmask : x.x.1111 1111. 0000 0000 = 255.0

Jawab:

1. Network ID

Network ID = IP Address AND Netmask

IP address : x.x.0010 1101. 0000 0001

Netmask    : x.x.1111 1111. 0000 0000

——————————————————

AND           : x.x.0010 1101. 0000 0000 = 45.0

Sehingga Network ID nya adalah 192.168.45.0

2. Broadcast ID

Broadcast ID = IP Address OR ~Netmask

IP address : x.x.0010 1101. 0000 0001

~Netmask  : x.x.0000 0000 . 1111 1111

———————————————————

OR : x.x.0010 1101. 1111 1111 = 45.255

Sehingga Broadcast Id nya adalah 192.168.45.255

3. Range IP

Range IP = Network ID hingga(-) Broadcast ID

Sehingga Range IP nya adalah 192.168.45.0 – 192.168.45.255

4. Jumlah Host

Jumlah Host = [tex]2^{y} -2[/tex]

dimana y adalah jumlah biner 0 pada octet 3 dan 4 netmask

/24 bila di-binerkan menjadi:

1111 1111. 1111 1111. 1111 1111. 0000 0000

255         255        255            0

Sehingga Jumlah Host

= [tex]2^{8} - 2[/tex]

= [tex]256 - 2[/tex]

= [tex]254[/tex]

39. Tentukan Host ID dan Net ID dari IP address 10.10.10.1​


Host ID dan Net ID adalah bagian-bagian dari alamat IP yang digunakan untuk membedakan antara perangkat atau host dalam jaringan dengan jaringan itu sendiri.

Untuk IP address 10.10.10.1, dapat ditentukan sebagai berikut:

Net ID: 10.10.10Host ID: .1

Net ID merupakan bagian dari alamat IP yang digunakan untuk menentukan jaringan yang digunakan, dan Host ID merupakan bagian dari alamat IP yang digunakan untuk menentukan perangkat yang terhubung ke jaringan tersebut.

Net ID dapat digunakan untuk menentukan subnet dari jaringan, sementara Host ID digunakan untuk menentukan alamat unik dari perangkat yang terhubung ke jaringan tersebut.
