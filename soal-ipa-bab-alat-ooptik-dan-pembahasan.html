OPTIKA
• O. Geometri
→ pembiasan cahaya

n₁ = 1,5. f₁ = 20 cm
n₂ = 1,4. f₂ = __?

persamaan pembuat lensa
1/f = (nₗ/nₘ - 1) (1/R₁ + 1/R₂)

untuk m = udara → nᵤ = 1
1/f = (nₗ/nᵤ - 1) (1/R₁ + 1/R₂)
1/f = (nₗ - 1) (1/R₁ + 1/R₂)

f berbanding terbalik dengan (n - 1)

Jika (n-1) menjadi (1,4-1)/(1,5-1) = ⅘ kali, jarak fokus menjadi ⁵/₄ kali semula atau ⁵/₄ × 20 = 25 cm ✔️


atau
1/f₁ / 1/f₂ = [(n₁ - 1) (1/R₁ + 1/R₂)] / [(n₂ - 1) (1/R₁ + 1/R₂)]
f₂ / f₁ = (n₁ - 1) / (n₂ - 1)
f₂ / 20 = (1,5 - 1) / (1,4 - 1)
f₂ / 20 = 0,5 / 0,4
f₂ / 20 = ⁵/₄
f₂ = ⁵/₄ × 20
f₂ = 25 cm ✔️

4. Bab alat optik kls 11


step 1

__ __ __

fok = fob/M

fok = 40/8

fok = 5 cm

step 2

__ __ __

d = fob + fok

• ALAT OPTIK

→ teropong panggung (teropong tonil)

Diketahui :

Perbesaran teropong (M) = 8 kali

fokus objektif (fob) = 40 cm

Kondisi pengamatan = tanpa berakomodasi

Ditanya :

panjang teropong panggung (d) ....?

Penjelasan :

Rumus perbesaran teropong panggung.

Bila berakomodasi

M = fob/fok

Bila tak berakomodasi

M = fob/sok

Sehingga, gunakan rumus untuk mata tanpa akomodasi.

M = fob/sok

8 = 40/sok

sok = 40/8

sok = 5 cm

Lalu, rumus untuk mencari panjang teropong tonil yaitu:

Bila berakomodasi

d = fob + fok

Bila tidak berakomodasi

d = fob + sok

Gunakan rumus panjang untuk mata tidak berakomodasi.

d = fob + sok

d = 40 cm + 5 cm

d = 45 cm

Maka, panjang teropong tonil tersebut adalah 45 cm.

Opsi di soal salah, karena fokus dari objektifnya saja sudah 40 cm, maka seharusnya panjang teropong pasti akan lebih dari 40 cm.

11. Rumus Alat-Alat Optik dan contoh soal


Alat Optik: Teropong Bintang (Astronomi) :

Perbesaran teropong bintang untuk mata berakomodasi maksimum dirumuskan sebagai berikut.

M=fob/sok

Keterangan:

M = perbesaran teropong

sok = jarak benda lensa okuler

fob = jarak fokus lensa objektif

Sedangkan panjang teropong bintang untuk penggunaan mata berakomodasi maksimum dirumuskan sebagai berikut.

d = s’ob + sok

d = fob + sok

Keterangan:

d = panjang teropong

s’ob = jarak bayangan lensa objektif

sok = jarak benda lensa okuler

fob = jarak fokus lensa objektif

31. Bab Alat optik kls 11


• ALAT OPTIK

→ teropong bintang

Diketahui :

Perbesaran (M) = 15 kali

jarak lensa objektif dan okuler atau biasa disebut panjang teropong (d) = 60 cm

jarak benda lensa objektif (sob) = 15 m

Ditanya :

s'ob = ___?

Penjelasan :

Dikatakan di soal bahwa mata tidak berakomodasi. Sehingga rumus perbesaranya,

M = fob/fok

15 = fob/fok

15 fok = fob

fok = 1/ 15 fob

Dan, rumus panjang mikroskop untuk mata tidak berakomodasi,

d = fob + fok

60 = fob + 1/15 fob

60 = 15/15 fob + 1/15 fob

60 = 16/15 fob

fob = 15/16 × 60

fob = 225/4 cm

Sehingga untuk mencari bayangan pada lensa objektif, kita tinggal memasukkan angka angka yang ada. Ingat bahwa satuan sob dalam soal adalah 15 meter. Ubah menjadi 1500 cm agar satunya sama.

1/fob = 1/sob + 1/s'ob

1/(225/4) = 1/1500 + 1/s'ob

4/225 - 1/1500 = 1/s'ob

77/4500 = 1/s'ob

s'ob = 58,44 cm

Sehingga, jarak bayangan pada lensa objektif sebesar 58,44 cm. Sehingga opsi yang tepat adalah d.

step 1

__ __ __

M = fob/fok

fob = M•fok

fob = 15fok

step 2

__ __ __

fob + fok = d

[15fok] + fok = 60

16fok = 60

fok = 60/16 = 15/4

step 3

__ __ __

fob = 15fok =

15•15/4 = 225/4 cm

step 4

__ __ __

1/s'ob = 1/fob - 1/sob

1/s'ob = 1/[225/4] - 1/1500

1/s'ob = 4/225 - 1/1500

1/s'ob = 80/4500 - 3/4500 = 77/4500

s'ob = 4500/77

s'ob = 58,44 cm ✔

39. soal tentang IPA bab tekanan


Tekanan

P = F/A
P = (m x g)/(p x l)
P = (40 x 10)/(1 x 0,4)
P = 400/0,4
P = 1000 N/m^2
Jadi besar tekanan kardus terhadap lantai adalah 1000 N/m^2.

Keterangan:
P = tekanan (satuan N/m^2)
F = gaya (satuan N)
A = luas bidang tekan (satuang m^2)
m = massa (satuan kg)
g = percepatan gravitasi (satuan m/s^2)
p = panjang (satuan m)
l = lebar (satuan m)
