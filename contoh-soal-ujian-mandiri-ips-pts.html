29. tolong bantu ya soal ujian mandiri


ax² + 5x - 12 = 0

Salah satu akar persamaannya adalah 2, jadi:


(ax + .......) (x - 2) = 0 ⇒ salah satu akar persamaannya adalah 2 ( yaitu x-2)


Isian titik-titik tersebut adalah:


...... di kali -2 = -12 ⇒ jadi titik-titik tersebut adalah 6.


Jadi: (ax + 6) (x - 2) = 0


Untuk mencari nilai a adalah:


(ax dikali -2) + (6 dikali x) = 5x

-2ax + 6x = 5x

-2a + 6 = 5

-2a = -1

a = 1/2



Jadi ⇒ (1/2x + 6) (x - 2) = 0


Akar persamaan yang lainnya adalah:


(1/2x + 6) = 0

1/2x = -6

x = -12


Jawaban A

33. rumus pola bilangan aritmatika suku ke n?? sekalian contoh soal nya plisss soalnya besok buat ujian PTS......​


Rumus pola bilangan aritmatika ke n :

Un = a + ( n - 1 ) . b

a = suku awal

b = beda ( u2 - u1 )

n = suku ke n

Contoh soal :

1. pada pola bilangan aritmatika 3 , 5 , 7 , 9 , ..... maka suku ke 8 adalah ..

Dik :

a = 3

b = 2

n = 8

Dit :

U8 = ?

Jawab :

Un = a + ( n - 1 ) . b

U8 = 3 + ( 8 - 1 ) . 2

= 3 + 7 . 2

= 3 + 14

= 17

2. suku ke tujuhbelas dari barisan : 98,92,86,80,.... adalah ....

Dik :

a = 98

b = - 6

n = 17

Dit :

U17 = ?

Jawab :

Un = a + ( n - 1 ) . b

= 98 + ( 17 - 1 ) . - 6

= 98 + 16 . - 6

= 98 + - 96

= 2

3. Rumus suku ke n barisan : 22,19,16,13,10,.... adalah ....

Dik :

a = 22

b = - 3

Dit :

rumus suku ke n = ?

Jawab :

Un = a + ( n - 1 ) . b

= 22 + ( n - 1 ) .- 3

= 22 - 3n + 3

= 25 - 3n
